# Interactive Digital Counter

A simple interactive digital counter built using HTML, CSS, and JavaScript.

This project demonstrates the fundamentals of DOM manipulation, event handling, and dynamically updating the user interface with JavaScript.

---

## Features

* Increase counter value
* Decrease counter value
* Reset counter to zero
* Dynamic color changes:

  * Green for positive numbers
  * Red for negative numbers
  * Black for zero

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## Project Structure

```plaintext
project-folder/
│
├── index.html

└── script.js
```

---

## How It Works

The application uses JavaScript to:

* Select HTML elements using `document.querySelector()`
* Listen for button clicks with `addEventListener()`
* Update the displayed counter using `textContent`
* Dynamically change text color based on the counter value

---

## Learning Objectives

This project helps practice:

* Connecting JavaScript files to HTML
* Understanding `let` vs `const`

* Event listeners
* Updating UI dynamically
* Basic conditional statements

## Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/interactive-counter.git
```

2. Navigate into the project folder:

```bash
cd interactive-counter
```

3. Open `index.html` in your browser.

No frameworks. No build tools. No unnecessary suffering.
